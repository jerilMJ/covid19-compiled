# covid19-compiled

![last updated badge](https://img.shields.io/github/last-commit/jerilMJ/covid19-compiled?label=last%20updated&style=flat-square)

Provides latest country-wise data from CSSEGISandData/COVID-19 all compiled into a JSON file.
Inspired by [this project.](https://github.com/pomber/covid19)

## Sample json response:

```js
  "JPN": {
    "name": "Japan",
    "date": "2020-04-13",
    "confirmed": 7370,
    "confirmed_diff": 622,
    "recovered": 784,
    "recovered_diff": 22,
    "deaths": 123,
    "deaths_diff": 15
  },
```
